# COVID-19 Dashboard (Tableau)



## Overview
This project visualises global COVID-19 data using interactive dashboards created in Tableau. The goal was to transform raw COVID-19 data into clear visual insights that make it easier to understand trends in cases, deaths, and vaccination progress across different regions.

The dashboard was built using data prepared with SQL queries and visualised in Tableau.

---

## Objectives
- Visualise global COVID-19 trends
- Compare infection and death rates across countries
- Present data in an interactive and easy-to-understand format
- Practice building data dashboards using Tableau

---

## Dashboard Features
The dashboard includes visualisations such as:

- Global COVID-19 case totals
- Total deaths by country
- Percentage of population infected
- Geographic distribution of cases using maps
- Trend analysis over time

These visualisations allow users to quickly identify patterns and compare the impact of COVID-19 across different countries.
