# Programming & Data Projects Portfolio

This repository contains projects I have completed while studying Computer Science and developing practical programming and data analysis skills. The projects involve working with real datasets and applying tools such as SQL, Python, and data visualisation software to perform data cleaning, exploration, and analysis.

The goal of this portfolio is to document practical work and continuously add new projects as I develop my technical skills.

---

## Projects

### SQL Data Cleaning – Nashville Housing
This project focuses on cleaning a real housing dataset using SQL.

Key techniques used:
- Standardising date formats
- Handling missing values using `ISNULL`
- Splitting address fields using `SUBSTRING`, `CHARINDEX`, and `PARSENAME`
- Converting categorical values using `CASE`
- Identifying duplicate records using `ROW_NUMBER` and Common Table Expressions (CTEs)

Folder: `sql-cleaning-nashville-housing-data`

---

### SQL Data Exploration – COVID Dataset
This project explores global COVID-19 data using SQL queries to identify trends and calculate metrics.

Key techniques used:
- Aggregate functions
- Joins
- Window functions
- Temporary tables
- Creating SQL views

Folder: `sql-covid-data-exploration`
